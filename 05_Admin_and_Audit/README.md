#### STEP File Types Explained
- **`STEP_Converted/`** — These are the OpenSCAD mesh geometry converted to STEP format via Zoo's CAD engine. They preserve exact OpenSCAD dimensions but are tessellated (faceted surfaces). Best for: dimensional verification, import reference.
- **`STEP_Parametric/`** — These were generated by Zoo's Text-to-CAD ML from engineering prompts. They are true B-rep solids with NURBS surfaces. Best for: production tooling, CMM inspection, downstream CAD operations.

## How to Use This Package

### For Production CAD (SolidWorks / Fusion 360 / NX / Creo)
1. Import `STEP_Parametric/*.step` files as starting geometry
2. Import `STEP_Converted/*.step` files for dimensional reference
3. Use `parameters.json` + drawings for all dimensions/tolerances
4. Rebuild parametric features in your native CAD system
5. The FreeCAD scripts in `A_CAD_Source/freecad_scripts/` show the parametric intent and feature tree structure

### For 3D Printing / Prototyping
1. Use `STL/*.stl` files directly
2. Print `E_Fit_Coupons/*.stl` first to validate interface fit
3. Recommended: PA12 SLS for functional prototypes

### For Injection Mold Tooling
1. Use `STEP_Parametric/*.step` as mold cavity reference
2. Drawings in `C_Drawings/` contain all GD&T for tool qualification
3. Draft angles (0.5°) and minimum wall thickness (2.5mm) specified in drawings
4. Material spec: PA66-GF30 (Zytel 70G30HSL or equivalent)
