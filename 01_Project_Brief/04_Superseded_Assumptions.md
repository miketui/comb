# Superseded Assumptions

The following assumptions from prior packages are no longer valid:

- Multiple comb heads inherit a universal male tang → **WRONG: only TC-001 has a tang**
- Only handles are receiver parts → **WRONG: all 5 attachments are receivers**
- The lock mechanism can live in the Main Comb → **WRONG: lock is in each attachment**
- TC-002 and TC-003 are "comb heads" with their own tangs → **WRONG: they are accessories**
- Fit coupons should be built around many male comb heads → **WRONG: one tang family only**
- Handle diameters from original drawings are final → **WRONG: TH-002 and TH-003 need thickening for comfort**
- Tooth geometry from original STLs is production-ready → **WRONG: root fillets and taper required**
- Crown peaks are acceptable as-is → **WRONG: need ≥1.5mm fillet radius**
- Force targets from either prior package alone are correct → **MERGED: 4-8N insertion, ≥20N pullout**

## Corrected master rule

```
- TC-001 Main Comb is the only male connector (passive tang + notch)
- All 5 attachments are female receivers with identical lock cores
- Lock hardware lives in each attachment, not in TC-001
- Magnets are optional secondary alignment support only
- All teeth require root fillets ≥0.5mm and tapered profiles
- All handles require grip texture and minimum comfort dimensions
- All previous outputs based on multiple male comb heads are superseded
```
